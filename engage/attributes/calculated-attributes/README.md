# Calculated Attributes

In Sortment, **calculated** Attributes are reusable logic definitions that classify users based on behaviors, characteristics, or transactions.&#x20;

These Attributes are ideal for:

* Complex segmentation needs
* Aggregations (e.g., total orders, spend)
* Business calculations that depend on multiple columns

They can be reused across Profiles, Audiences, and Analytics to maintain consistent business logic.

***

## Creating a calculated attributes using AI

Sortment provides an **AI-powered creation option** that makes building calculated attributes fast and approachable—even without technical expertise.

### Open the attributes section:

1. Click the **star-shaped icon** (`✶`) in the left or top navigation bar.
2. This opens the attributes page, where you can view, manage, and create new attributes.

<figure><img src="../../../.gitbook/assets/image (146).png" alt=""><figcaption></figcaption></figure>

### Step 1: Click New attribute

Start by clicking the **“+ New Attribute”** button at the top right of the Attributes page. Choose calculated attributes.

Select "Use AI" as the creation method

### Step 2: Define the use case

* **Describe your use-case**: What are you trying to identify (e.g., “Number of times a customer bought premium products”)?
* **Outline examples or scenarios** _(optional but helpful)_: Provide additional context or example on how the calculation may be performed.

### Step 3: Generate the logic

After submitting your prompt, Sortment generates the logic for your attributes. It will also create a name and description for the attribute.

* You can review and refine using additional instructions like “exclude users with returns.”

### Step 4: Preview results

Click **“Preview results”** to validate that the attribute logic works as intended. This ensures data accuracy before you publish.

### Step 5: Finalise details

* The attribute name and description is generated by AI. Review the details and update if needed.
* Optionally add **tags** for searchability and organisation.

### Step 6: Save the attributes

Click **“Finish”** to save the attributes. It’s now available for use across Audiences, Profiles, and Campaign logic.
