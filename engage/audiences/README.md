# Audiences

Segmenting users to target with personalised campaigns is an effective campaign strategy. Currently, these workflows involve help from data team internally to generate a CSV file manually from data in your warehouse.

Sortment allows your to create audiences on top of your warehouse data using AI and no-code visual builder, to activate data directly from your warehouse. It enables marketers to use data directly from business’ single source of truth—the data warehouse.

Segmenting your users to target them with personalised campaigns is a highly effective strategy. Currently, this process is involved with manual CSV generation from your data warehouse, often requiring assistance from an internal data team.

Sortment revolutionizes this by allowing you to create audiences directly on top of your warehouse data using a no-code visual builder. This enables marketers to activate data directly from your business's single source of truth—the data warehouse.

Sortment executes an SQL query on your warehouse under the hood, but the no-code interface allows anyone to create these audiences and make data access more democratic.

You can create audience using the AI-powered prompts via _Ask AI_ to generate audiences instantly, using visual builder, and also use the SQL builder with AI enhanced capabilities to create audiences (or migrate existing audiences to Sortment).

#### **How it Works:**

Under the hood, Sortment executes an SQL query on your warehouse. However, our intuitive no-code interface democratizes data access, empowering anyone to create sophisticated audiences without needing SQL expertise.

**The easiest way to create audiences: Just ask Rho!**

While you can create audiences using our visual builder or even our SQL builder with AI-enhanced capabilities (perfect for migrating existing audiences), the most efficient way to build an audience is by **simply asking Rho.**

**To create an audience, just tell Rho what you need.** For example, you can say:

* "Rho, create an audience of customers who have made a purchase in the last 30 days and are located in California."
* "Rho, I need an audience of users who have signed up but haven't completed their profile."
* "Rho, create an audience of high-value customers who have spent over $500 in the last quarter."

Rho will take care of the rest, translating your request into the necessary audience parameters and building it for you.&#x20;
