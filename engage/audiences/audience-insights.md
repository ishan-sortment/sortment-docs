# Audience Insights

### **Overview**

**Audience Insights** enhance audience creation by providing a clearer, data-driven perspective. This allows marketers to confidently validate and optimize their audiences for more effective, targeted campaigns. With built-in data exploration and validation tools, marketers can ensure the accuracy of their segments, reduce the need for analyst intervention, and achieve greater efficiency in campaign execution.

### **Generating Insights**

Insights are updated whenever refresh button is clicked. This is kept action-driven to reduce load of queries on the warehouse.

#### **Audience Reachability**

Calculates percentage of users from audience reachable per channel, based on their global reachability status.

#### **Audience Breakdown**

Create a data distribution bar chart for users based on selected columns, with options for expanded view and further analysis for each chart.

#### **Audience Overlap**

Compare the current audience with other saved audiences and see overlap in the user count. Audiences then can be used as filters to keep or remove these overlapping users in the new audience.
