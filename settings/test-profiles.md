# Test Profiles

Test profiles are user records with contact fields (e.g., email, phone number) used **to simulate and** [**test messaging campaigns**](../engage/campaigns/building-a-campaign.md#step-5-testing-your-campaign) before going live. They help ensure the correct message formatting, personalization, and delivery across channels like Email, SMS, and WhatsApp.

This is located in Workspace Settings → Test Profiles.

***

#### Setup

1. **Download Sample file or Current List**:
   * In case the test profiles are not set up yet, you will see an option to download the sample file. Else, click on **"Download current list"** button to get a currently uploaded test profiles in a CSV file.
2. **Update the list**:
   * Open the CSV and input your test contact fields and values.
   * Save the file and upload it by clicking **"Update list"**.
   * Once uploaded, these test profiles are available for quick access to send test communciations to internal users in campaigns, templates, or journeys.

***

#### Managing test profiles

* **Updating test profiles** automatically replaces the existing list.
* There is no version history — make sure to keep a local copy before changes.
* To reset or clear test profiles:
  * Simply upload a new CSV with blank or updated data.
  * There is no explicit "delete" function, but overwriting the list achieves the same effect.

***

**Use of test profiles**

* Before sending a[ **campaign**](../engage/campaigns/building-a-campaign.md#testing-your-campaign), test profiles let you simulate sending the campaign without reaching real contacts.
* You can preview how personalization (e.g., names, custom fields) renders across all channels.
* This helps ensure the **accuracy of logic, formatting, and content** before the actual send.
* Test sends using these profiles help validate journeys, triggers, and template behavior under real-world conditions—without risk.
