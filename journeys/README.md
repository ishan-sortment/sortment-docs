# Journeys

### Journey overview

Journeys in Sortment are real-time, behavior-based flows that allow you to engage users with the right message at the right time across channels. They help you orchestrate marketing strategies by triggering actions based on real-time user behavior.

With Sortment Journeys, you can:

* Automate personalised user experiences
* Trigger messages when specific actions are performed by the user
* Create goal oriented complex workflows
* Connect with users through multiple channels (Email, SMS, WhatsApp)

Sortment Journeys are great for:

* Onboarding new users
* Sending promotions based on user interest
* Recovering abandoned checkouts
* Re-engaging inactive users
* Running personalised campaign flows

***

### Required permissions

Before you can create or manage journeys in Sortment, ensure the following setup and permissions are in place:

* [**Warehouse setup**](../setup/data-warehouses/): Sortment requires a connected warehouse to store and access real-time user data and events during journeys.
* [**Channel configuration**](../setup/communication-channels/): Messaging channels (Email, SMS, WhatsApp) must be set up.&#x20;
* [**Event tracking**](../schema/setting-up-events-table.md): User events (e.g., product\_viewed, cart\_abandoned) must be created on Sortment and available to trigger journey actions.
