# Getting Started

To begin working with journeys in Sortment, follow these steps:

### 1. Navigate to the journeys section

From your Sortment dashboard, click on the **Journeys** tab in the top navigation bar.

This opens the **Journeys page**, where you can:

* View all existing journeys and their current status (e.g., Stopped, Running)
* See who created or last modified each journey
* Check the number of users who have entered or converted within each journey
* Sort and search for journeys using the search bar
* Toggle visibility of archived journeys

### 2. Create a new Journey

To start building a new journey:

* Click the **New journey** button in the top right corner.
* You’ll be redirected to the [Journey Builder](journey-builder.md)**.** Follow the steps in the linked document to create the journey.

### 3. Understand the status indicators

Each journey shows a status label:

* **Stopped**: The journey is inactive and not receiving new user entries.
* **Running** _(if implemented)_: Indicates the journey is live and actively responding to triggers.

You’ll also be able to view:

* **Created date and creator**
* **Last modified date and editor**
* **Entry and conversion metrics** (once live data starts flowing)

### 4. Organise and manage journeys

Sortment allows you to:

* Duplicate existing journeys for quick iteration
* Archive journeys you no longer use (via the menu on the right of each row)
* Maintain visibility and management with sorting, filtering, and searching options
