# Data Warehouses

A data source is the central warehouse where your organization's data lives. To add a new data source to your Sortment workspace, follow these steps:

1.  Select the source you want to add to Sortment.<br>

    <figure><img src="../../.gitbook/assets/image (89).png" alt=""><figcaption></figcaption></figure>
2. Enter the relevant credentials. These differ depending on the source. Refer to the relevant source documentation for further details.
3. Click Continue. Before continuing, Sortment validates that the credentials shared are correct and it has the necessary access and permissions. This step can take several seconds. If the test fails, you need to confirm and re-enter your credentials.
4. Once finished, this will appear in Data connection under Settings page. You can use it set up Schema on Sortment.
