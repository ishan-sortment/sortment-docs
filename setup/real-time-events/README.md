# Real-time Events

Real-time Events are customer actions, like a purchase or app interaction, that can be used to trigger an action to a user or update their profile data. These are ideally sent from an event collection app like Rudderstack, Segment etc to Sortment. This setup is independent on warehouse connection.

There are three steps to connect real-time events to Sortment:

* Setup event source on Sortment
* Connect event source with your event source
* Whitelist event payload
