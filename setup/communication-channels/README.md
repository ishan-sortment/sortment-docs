# Communication Channels

Communications house the channels and providers that you can integrate with Sortment to send communications.&#x20;

Get Started:

1. Choose the channel that you want to configure&#x20;
2. Chose the provider that you want to connect

<figure><img src="../../.gitbook/assets/Screenshot 2024-01-11 at 6.43.06 PM.png" alt=""><figcaption></figcaption></figure>

1. Add relevant fields to complete connection

**Adding more providers**

Once you have set up your first channel, you can add more providers to those channels by adding within selected channels.&#x20;
