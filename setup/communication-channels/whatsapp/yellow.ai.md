# Yellow.ai

## Overview

Sortment lets you send broadcast messages via your existing Yellow.ai account. It is required to set up / have an account before configuring the same on Sortment.&#x20;

### Set up your account on Sortment

1. Navigate to channels. Click **Add channel**
2. Under WhatsApp channel, select **Yellow.ai** as the provider
3. Setup an integration name (_only alphanumeric allowed, no special characters or spaces_)
4. You need following credentials from Yellow.ai (follow steps below)
   1. Bot Id
   2. API Key
   3. Sender

### Configure the Integration

You can get the following details from Yellow.ai account through these steps:

1. **Bot Id**: To obtain the bot ID, you can click on the web link from any page of the bot after creating it and copy the 14-digit alphanumeric value (e.g., x16763649\*\*). The bot ID is different in each environment.

<figure><img src="https://files.buildwithfern.com/fyno.docs.buildwithfern.com/docs/2025-03-18T12:11:21.833Z/images/whatsapp_yellow_botid.png" alt=""><figcaption></figcaption></figure>

*   **API Key**: You can access API Key from your Yellow.ai account. Go to _Bot Settings -> API Keys_ . If there are no API keys, you can create one by clicking ‘Generate API key’

    <figure><img src="https://files.buildwithfern.com/fyno.docs.buildwithfern.com/docs/2025-03-18T12:11:21.833Z/images/whatsapp_yellow_apikey.png" alt=""><figcaption></figcaption></figure>
* **Sender**: Choose the WhatsApp business number from which you want to send the campaign

### Message Delivery Status <a href="#message-delivery-status" id="message-delivery-status"></a>

Yellow.ai, by default, provides the status of the WhatsApp messages without any additional setup needed.
