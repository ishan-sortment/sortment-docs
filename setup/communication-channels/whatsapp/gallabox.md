# Gallabox

## Overview

Sortment lets you send broadcast messages via your existing Gallabox account. It is required to set up / have a Gallabox account before configuring the same on Sortment.&#x20;

### Set up your account on Sortment

1. Navigate to channels. Click **Add channel**
2. Under WhatsApp channel, select **Gallabox** as the provider
3. You need following credentials from Gallabox (follow steps below)
   1. Access token

## Creating access token in Gallabox

1. From the left navigation go to **Integration** in your Gallabox and Click on **Fyno** from the available apps.
2. Click on Connect. Provide a name for the integration and click on **Submit**. This will create a connection on Gallabox.
3. Select the WhatsApp Channel you want to connect with Fyno to get notifications on Gallabox and then update configuration.
4. Update the configuration with following **Status URL**: https://callback.fyno.io/?fyno-provider=gallabox
5. Once you update the configuration on Gallabox, an access token will be generated. Copy the **Access Token** and add to Sortment.&#x20;
6. Click Save to setup the integration on Sortment.
7.  Once done, click **Update Configuration** once again on Gallabox. The integration is now ready to use.
